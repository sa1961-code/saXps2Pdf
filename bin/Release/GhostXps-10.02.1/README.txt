== GhostXPS ==

Gxpsdll32.dll and gxpsdll64.dll are builds of the GhostXPS document 
conversion engine for Microsoft Windows. 

GhostXPS is distributed under the terms of the GNU GPL. See the file
COPYING for details. Source code and other binaries can be obtained
from http://ghostscript.com/

GhostXPS, GhostPCL, Ghostscript and the urw font set are also available
for commercial licensing under other terms. Please visit
http://artifex.com/ for more information.
